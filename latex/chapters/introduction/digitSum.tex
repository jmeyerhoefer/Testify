\subsection{Example 1 - \texttt{digitSum}} %%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%


\paragraph{Task a)} Implement a method \lstinline[basicstyle=\ttfamily]|digitSum: Nat -> Nat|,
that takes a natural number \lstinline[basicstyle=\ttfamily]|n| as input and returns its digit sum. Examples:
\begin{center}\begin{multicols}{2}
	\lstinline[basicstyle=\ttfamily]|digitSum 0N = 0N|\\
	\lstinline[basicstyle=\ttfamily]|digitSum 123N = 6N|\\
	\lstinline[basicstyle=\ttfamily]|digitSum 1234N = 10N|\\
	\lstinline[basicstyle=\ttfamily]|digitSum 4711N = 13N|
\end{multicols}\end{center}
