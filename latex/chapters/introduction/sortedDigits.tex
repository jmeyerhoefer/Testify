%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%


\paragraph{Task b)} Implement a method \lstinline[basicstyle=\ttfamily]|sortedDigits: Nat -> bool|,
that takes a natural number \lstinline[basicstyle=\ttfamily]|n| as input and returns
\lstinline[basicstyle=\ttfamily]|true| if the digits are in ascending order and \lstinline[basicstyle=\ttfamily]|false| otherwise. Examples:
\begin{center}\begin{multicols}{2}
	\lstinline[basicstyle=\ttfamily]|sortedDigits 0N = true|\\
	\lstinline[basicstyle=\ttfamily]|sortedDigits 5N = true|\\
	\lstinline[basicstyle=\ttfamily]|sortedDigits 159N = true|\\
	\lstinline[basicstyle=\ttfamily]|sortedDigits 1111N = true|\\
	\lstinline[basicstyle=\ttfamily]|sortedDigits 1101N = false|\\
	\lstinline[basicstyle=\ttfamily]|sortedDigits 543N = false|
\end{multicols}\end{center}
