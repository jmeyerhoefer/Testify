\subsection{Example 1 - \texttt{digitSum}} %%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%


Implement a method \fsharp|digitSum: Nat -> Nat|,
that takes a natural number \fsharp|n| as input and returns its digit sum. Examples:
\begin{center}\begin{multicols}{2}
	\fsharp|digitSum 0N = 0N|\\
	\fsharp|digitSum 123N = 6N|\\
	\fsharp|digitSum 1234N = 10N|\\
	\fsharp|digitSum 4711N = 13N|
\end{multicols}\end{center}
