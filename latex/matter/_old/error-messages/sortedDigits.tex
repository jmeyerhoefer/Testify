\subsection{Example 2 --- \texttt{sortedDigits}} %%%%%%%%%%%%%%%%%%%%%%%%%%
%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%


Implement a method \fsharp|sortedDigits: Nat -> bool|,
that takes a natural number \fsharp|n| as input and returns
\fsharp|true| if the digits are in ascending order and \fsharp|false| otherwise. Examples:
\begin{center}\begin{multicols}{2}
	\noindent
	\fsharp|sortedDigits 0N = true|
	\fsharp|sortedDigits 5N = true|
	\fsharp|sortedDigits 159N = true|
	\fsharp|sortedDigits 1111N = true|
	\fsharp|sortedDigits 1101N = false|
	\fsharp|sortedDigits 543N = false|
\end{multicols}\end{center}
